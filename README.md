# NGTU Code Platform

Платформа для автоматизированной адаптивной генерации и проверки программных заданий с персональными кабинетами пользователей и анализом динамики освоения курса.

## Возможности

- регистрация и авторизация пользователей (преподаватель / студент)
- управление темами, задачами и вариантами формулировок
- создание и проведение тестов по группам
- проверка решений на Python по тест-кейсам
- просмотр ответов и аналитики по тестам

## Технологии

- **Backend:** Java 17, Spring Boot, Spring Data JPA, Spring Security, PostgreSQL
- **Frontend:** Angular 17 (standalone), TypeScript, RxJS

## Структура репозитория

- `backend/` — REST API и бизнес-логика
- `frontend/` — клиентское приложение

## Быстрый старт (Docker)

Требования: установленный Docker Desktop / Docker Engine.

Из корня проекта:

```bash
docker compose up -d --build
```

После запуска:

- Frontend: `http://localhost:4200`
- Backend API: `http://localhost:8080/api/auth`
- PostgreSQL: `localhost:5432` (`postgres/postgres`, DB: `ngtu_code_platform`)

Тестовые пользователи:

- Teacher: `teacher` / `teacher123`
- Student: `student` / `student123`

Остановить и удалить контейнеры:

```bash
docker compose down
```

Полная очистка вместе с данными БД (volume):

```bash
docker compose down -v
```
